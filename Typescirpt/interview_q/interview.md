# Practice Goal

- ✅ Avoid using `any`
- ✅ Use `unknown` when needed
- ✅ Prefer `interface` for object contracts
- ✅ Use `type` for unions
- ✅ Write reusable Generics
- ✅ Enable strict mode
- ✅ Use Utility Types effectively
- ✅ Build type-safe applications

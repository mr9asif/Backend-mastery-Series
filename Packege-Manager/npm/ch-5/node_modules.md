# 🤔 node_modules কী?

`node_modules` হলো একটি Folder যেখানে Project-এর **সব Installed Packages এবং তাদের Dependencies** সংরক্ষিত থাকে।

সহজভাবে বললে,

> **তুমি npm দিয়ে যেসব Package Install করো, সেগুলো node_modules Folder-এর ভিতরে রাখা হয়।**

---

# 🧠 সহজভাবে বুঝো

ধরো তুমি একটি Restaurant খুলেছো।

তোমার কাছে একটা Shopping List আছে।

```
- Chicken
- Rice
- Oil
- Salt
```

এটা হলো `package.json`

এখন Market থেকে সব জিনিস কিনে Kitchen-এ এনে রাখলে।

Kitchen = `node_modules`

অর্থাৎ,

- `package.json` → কী কী লাগবে
- `node_modules` → আসলে কী কী Install হয়েছে

---

# কীভাবে তৈরি হয়?

যখন চালাও

```bash
npm install
```

npm

- `package.json` পড়ে
- Registry থেকে Package Download করে
- `node_modules` Folder তৈরি করে
- সব Package সেখানে Store করে

---

# ভিতরে কী থাকে?

Example

```bash
npm install express
```

Folder Structure

```text
node_modules/

├── express/
├── accepts/
├── body-parser/
├── bytes/
├── debug/
├── mime/
├── cookie/
└── ...
```

এখানে শুধু `express` না,

Express-এর যেসব Package লাগে সেগুলোও Install হয়।

এগুলোকে বলে **Dependencies**।

---

# কেন এত বড় হয়?

ধরো তুমি শুধু Install করলে

```bash
npm install express
```

তুমি ভাবছো ১টা Package Install হয়েছে।

আসলে

```
Express
      │
      ├── body-parser
      ├── accepts
      ├── cookie
      ├── send
      ├── mime
      └── ...
```

একটা Package-এর ভিতরে অনেক Dependency থাকে।

তাই `node_modules` অনেক বড় হয়ে যায়।

---

# npm কীভাবে ব্যবহার করে?

যখন লিখো

```js
import express from "express";
```

Node.js প্রথমে খুঁজে দেখে

```
node_modules/express
```

পেয়ে গেলে

Express Load করে।

---

# Module Resolution (Overview)

ধরো

```js
import express from "express";
```

Node.js

```
Current Folder
      │
      ▼
node_modules
      │
      ▼
express
      │
      ▼
Load Package
```

যদি Package না পায়

তাহলে Error দেয়।

```
Cannot find module 'express'
```

# Delete করলে কী হবে?

Delete করলে Project Run করবে না।

কারণ Installed Package থাকবে না।

তবে চিন্তার কিছু নেই।

আবার

```bash
npm install
```

চালালেই

সব Package আবার Download হয়ে যাবে।

---

# Git-এ Push করবো?

❌ না।

কখনোই না।

কারণ

- Folder অনেক বড়
- হাজার হাজার File থাকে
- `package.json` থেকেই আবার Generate করা যায়

Git-এ Push করবে

```
✅ package.json

✅ package-lock.json

❌ node_modules
```

তাই `.gitignore`-এ লিখি

```gitignore
node_modules/
```

# Best Practices ✅

- `node_modules` Manual Edit করো না।
- Git-এ Push করো না।
- Delete হয়ে গেলে `npm install` চালাও।
- `.gitignore`-এ সবসময় `node_modules/` রাখো।

---

# Common Mistakes ❌

❌ `node_modules` GitHub-এ Push করা।

❌ ভিতরের File Edit করা।

❌ Delete হওয়ার পরে Panic করা।

❌ Package Copy করে Manual Paste করা।
